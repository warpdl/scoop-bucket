# Scoop Bucket for WarpDL

Official [Scoop](https://scoop.sh) bucket for [WarpDL](https://github.com/warpdl/warpdl) â€” a high-performance, multi-threaded download manager for Windows.

## Installation

```bash
scoop bucket add warpdl https://github.com/warpdl/scoop-bucket
scoop install warpdl
```

## Upgrade

```bash
scoop update warpdl
```


## Uninstall 

```bash
scoop uninstall warpdl
scoop bucket rm warpdl  # optional
```
